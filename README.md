# 📌 Discord Project Manager Bot

Bu proje, kullanıcıların projelerini Discord üzerinden yönetebilmesini sağlayan bir Discord botudur.  
Bot sayesinde projeler eklenebilir, güncellenebilir, silinebilir ve projelere beceriler atanabilir.

---

# 🚀 Özellikler

- Yeni proje ekleme
- Projeleri listeleme
- Proje bilgilerini güncelleme
- Projelere beceri ekleme
- Proje silme
- SQLite veri tabanı desteği
- Discord bot sistemi

---

# 🛠 Kullanılan Teknolojiler

- Python
- SQLite3
- Discord.py
